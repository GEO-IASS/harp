NPP_SUOMI_L2_CRIMSS_EDR_TEMPERATURE

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of NPP_SUOMI_L2_CRIMSS_EDR_TEMPERATURE data.

field name type dimensions unit description
datetime double {time} seconds since 2000-01-01 time of the measurement (in seconds since 2000-01-01 00:00:00)
latitude double {time} degree_north tangent latitude
longitude double {time} degree_east tangent longitude
sensor_azimuth_angle double {time} degree azimuth angle (measured clockwise positive from North) to Satellite at each retrieval position
sensor_zenith_angle double {time} degree zenith angle to Satellite at each retrieval position
solar_azimuth_angle double {time} degree azimuth angle of sun (measured clockwise positive from North) at each retrieval position
solar_zenith_angle double {time} degree zenith angle of sun at each retrieval position
temperature double {time, vertical} K temperature profile
pressure double {vertical} hPa pressure levels for temperature retrieval
index int32 {time}   zero-based index of the sample within the source product

Ingestion options

The table below lists the available ingestion options for NPP_SUOMI_L2_CRIMSS_EDR_REDR products.

option name legal values description
dataset h2o, temp, press the dataset of the L2 product to ingest; option values are ‘h2o’ (h2o mass mixing ratio vs pressure, this is the default), ‘temp’ (temperature vs pressure)’, ‘press’ (pressure vs altitude)

Mapping description

The table below details where and how each variable was retrieved from the input product.

field name mapping description
datetime path /All_Data/CrIMSS-EDR-GEO-TC_All/StartTime, /All_Data/CrIMSS-EDR-GEO-TC_All/MidTime
description the time converted from seconds since 1958-01-01 to seconds since 2000-01-01T00:00:00
latitude path /All_Data/CrIMSS-EDR-GEO-TC_All/Latitude
longitude path /All_Data/CrIMSS-EDR-GEO-TC_All/Longitude
sensor_azimuth_angle path /All_Data/CrIMSS-EDR-GEO-TC_All/SatelliteAzimuthAngle
sensor_zenith_angle path /All_Data/CrIMSS-EDR-GEO-TC_All/SatelliteZenithAngle
solar_azimuth_angle path /All_Data/CrIMSS-EDR-GEO-TC_All/SatelliteAzimuthAngle
solar_zenith_angle path /All_Data/CrIMSS-EDR-GEO-TC_All/SatelliteAzimuthAngle
temperature path /All_Data/CrIMSS-EDR_All/Temperature
pressure path /All_Data/CrIMSS-EDR_All/PressureLevels_Temperature
